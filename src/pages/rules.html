夏季清凉服务挑战赛

夏季清凉服务挑战赛活动规则

一、活动时间与参与资格

  • • 活动时间：2025年5月1日至2025年8月31日
  • • 参与资格：滴滴平台注册的有效司机

二、"有效空调相关差评"定义

  • • 定义：乘客对司机空调服务提出的有效投诉
  • • 判定来源：基于平台众判系统结果，判定为司机责任的空调相关投诉
  • • 差评类型：包括但不限于"空调不够凉"、"车内温度过高"、"拒绝开空调"等空调服务相关投诉

三、清凉服务荣誉值规则

  • • 初始值：活动开始时，每位司机初始荣誉值为100分
  • • 扣分规则：每收到1次有效空调相关差评，荣誉值扣减20分
  • • 最终用途：活动结束时，荣誉值90分及以上的司机将获得额外100服务分奖励，并有机会参与终极大奖抽奖

四、周期任务要求

  • • 周期定义：以自然周为单位，每周一0点至周日24点
  • • 任务要求：
  • 1. 周期内空调相关差评为0
  • 2. 周期内完成至少20单订单
  • • 达成方式：同时满足上述两个条件即视为达成周期任务

五、抽奖机会获取方式

  • • 完成周期任务：每完成一个周期的任务获得1次抽奖机会
  • • 邀请好友：每成功邀请1位司机参与活动获得1次抽奖机会
  • • 连续达成：连续2周达成周期任务，额外获得1次抽奖机会

六、奖品类型说明

  • • 现金奖励：¥1、¥2、¥5、¥88、¥188
  • • 服务分：50分、100分、200分
  • • 免佣卡：1小时免佣、2单免佣、5单免佣
  • • 平台积分：50积分、100积分

七、邀请好友规则

  • • 邀请对象：未参与本次活动的平台注册司机
  • • 邀请方式：通过活动页生成的专属二维码、链接或海报
  • • 成功定义：被邀请司机通过邀请人的专属链接成功报名参与活动

八、奖励发放方式与时间

  • • 抽奖奖励：即时发放至司机账户
  • • 连胜奖励：系统自动判定并即时发放
  • • 最终荣誉值奖励：活动结束后7个工作日内统一发放

九、作弊行为认定及处理

  • • 作弊行为包括但不限于：刷单、虚假邀请、利用系统漏洞获取抽奖机会等
  • • 一经发现作弊行为，平台有权取消参与资格并收回所有已发放奖励

常见问题

如何知道我收到了空调相关差评？

系统会通过App消息中心向您推送通知，您也可以在活动页面查看当前荣誉值和差评情况。

抽奖获得的奖励多久到账？

现金、服务分、积分、免佣卡等奖励均为即时发放，抽奖后即可使用。

活动期间可以累积多少次抽奖机会？

抽奖机会无上限，您可以通过完成周期任务、邀请好友等多种方式获取。
